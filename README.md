# draft

unaffiliated with wizards of the coast

# run

- [node.js](http://nodejs.org/)

- `make`

- `node app.js`

- <http://localhost:1337>

# updating

generally you can update with `git pull`; if that doesn't work,
rerun `make`; if that still doesn't work, please file an issue
